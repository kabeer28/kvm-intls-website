# KVM Intls Website

A modern, responsive website for KVM Intls, a heavy-duty truck and trailer parts distributor in the Greater Toronto Area.

## Features

- Responsive design for all device sizes
- Product catalog display
- Partner brand showcase
- Contact form with Gmail integration
- Smooth transitions and animations

## Technologies Used

- React
- React Router
- Tailwind CSS
- Lucide React Icons

## Getting Started

1. Clone the repository:
```
git clone https://github.com/kabeer28/kvm-intls-website.git
```

2. Install dependencies:
```
cd kvm-intls-website
npm install
```

3. Run the development server:
```
npm run dev
```
